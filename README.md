# Product Synthesis Rules (`product-synthesis-rules`)

本リポジトリは、自律型AIエージェント（Cline等）に対して、文化的・地域的・組織的な背景から企業のコアコンピタンスを解剖し、最適なプロダクト統合戦略を立案させるための厳格な制約（`.clinerules`）を提供するものです。

このルールの根底にある「他者の卓越性を認める審美眼」と「多層的コンピタンス・マトリクス」の哲学と概念的背景については、以下の記事を参照してください。
